# 📰 News & Events Elementor Widgets

A complete news and events management system with **custom Elementor widgets** for easy drag-and-drop functionality. Perfect for news websites, event venues, and community organizations using WordPress and Elementor.

## 🚀 **Quick Start**

### **Step 1: Install Plugin**
1. **Zip the folder:** `news-events-elementor-widgets`
2. **WordPress Admin** → **Plugins** → **Upload Plugin**
3. **Upload and activate**

### **Step 2: Add Widgets**
1. **Edit page in Elementor**
2. **Find widgets in General category:**
   - 📰 News Display & News Manager
   - 📅 Events Display & Events Manager
3. **Drag and drop** to your page

### **Step 3: Start Adding Content**
- Use **Manager widgets** on admin pages to add content
- Use **Display widgets** on public pages to show content
- **Customize styling** with built-in Elementor controls
